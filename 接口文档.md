## 通用规范

### 请求格式

- 所有请求统一使用 **JSON** 格式
- 请求头需包含：`Content-Type: application/json`
- 需要认证的接口需在请求头携带：`Authorization: Bearer {token}`

### 响应格式

所有接口统一返回以下结构：

```json
{
  "code": 200,        // 业务状态码
  "message": "success", // 提示信息
  "data": {}          // 业务数据（可选）
}
```

### HTTP 状态码

| 状态码 | 含义 |
|--------|------|
| 200 | 请求成功 |
| 400 | 请求参数错误 |
| 401 | 未授权/Token失效 |
| 403 | 禁止访问 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |

## 认证相关

### 1. 用户注册

**接口描述**：新用户注册账号

- **请求方式**：`POST`
- **接口路径**：`/user/register`
- **是否需要认证**：否

#### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| username | string | 是 | 用户名，2-20个字符 |
| phone | string | 是 | 手机号码，11位 |
| password | string | 是 | 登录密码，至少6位 |

#### 请求示例

```json
{
  "username": "张三",
  "phone": "18340237677",
  "password": "123456"
}
```


#### 错误情况

| 状态码 | 错误信息 | 说明 |
|--------|----------|------|
| 400 | 用户名格式不正确 | 长度不在2-20范围内 |
| 400 | 手机号格式不正确 | 不符合11位手机号规则 |
| 400 | 密码长度不能少于6位 | 密码太短 |
| 409 | 该手机号已被注册 | 手机号已存在 |

### 2. 用户登录

**接口描述**：使用手机号和密码登录

- **请求方式**：`POST`
- **接口路径**：`/user/login`
- **是否需要认证**：否

#### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| phone | string | 是 | 手机号码 |
| password | string | 是 | 登录密码 |

#### 请求示例

```json
{
  "phone": "18340237677",
  "password": "123456"
}
```

#### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| userId | long | 用户ID |
| username | string | 用户名 |
| phone | string | 手机号 |
| avatar | string | 头像URL（可能为空） |
| token | string | 登录凭证 |

#### 响应示例

```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "userId": "10001",
    "username": "张三",
    "phone": "18340237677",
    "avatar": "",
    "token": "eyJhbGciOiJIUzI1NiIs..."
  }
}
```

#### 错误情况

| 状态码 | 错误信息 | 说明 |
|--------|----------|------|
| 400 | 手机号或密码错误 | 凭证不匹配 |
| 400 | 该用户不存在 | 手机号未注册 |

## 显著性检测模块

### 4. 视频上传模块

**接口描述**：获取当前登录用户的详细信息

- **请求方式**：`POST`
- **接口路径**：`/detect/upload`
- **是否需要认证**：是

#### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| userId | string | 是 | 用户ID |
| video | file | 是 | 需要进行显著性检测的视频 |
|frame|int|是|希望检测生成视频的帧率|
| video_type | string | 是 | 上传视频是2d还是3d|
| duration | int | 是 | 上传视频的时长，单位秒|

#### 请求示例

```json
{
  "userId": "112233",
  "video": "test.mp4",
  "frame": 30,
  "video_type": "3d"
}
```

#### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| video_avatar | string | 原视频URL |


#### 响应示例

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "video_avatar": "asregfdcvRWagfdVSASDF"
  }
}
```

## 错误码说明

### 通用错误码

| 错误码 | 含义 | 说明 |
|--------|------|------|
| 200 | 成功 | 请求处理成功 |
| 400 | 请求参数错误 | 参数格式不正确或缺少必填参数 |
| 401 | 未授权 | Token缺失、过期或无效 |
| 403 | 禁止访问 | 无权限执行此操作 |
| 404 | 资源不存在 | 请求的资源不存在 |
| 429 | 请求过于频繁 | 触发限流，请稍后重试 |
| 500 | 服务器内部错误 | 服务器发生错误 |

### 业务错误码

| 错误码 | 含义 | 说明 |
|--------|------|------|
| 1001 | 手机号已注册 | 注册时手机号已被使用 |
| 1002 | 手机号或密码错误 | 登录凭证不正确 |
| 1003 | 用户不存在 | 该手机号未注册 |
| 1004 | 原密码错误 | 修改密码时旧密码验证失败 |
| 2001 | 文件格式不支持 | 上传的视频格式不正确 |
| 2002 | 文件大小超限 | 视频文件超过500MB限制 |
| 2003 | 任务不存在 | 检测任务ID无效 |
| 2004 | 任务处理中 | 任务尚未完成，无法获取结果 |
| 2005 | 任务处理失败 | 检测任务执行失败 |

## 附录

### 视频格式支持

| 格式 | MIME类型 | 说明 |
|------|----------|------|
| MP4 | video/mp4 | 推荐格式 |
| AVI | video/x-msvideo | 支持 |
| MOV | video/quicktime | 支持 |
| MKV | video/x-matroska | 支持 |
| FLV | video/x-flv | 支持 |

### 存储限制

- 单个用户最多存储 **100个** 检测视频
- 单个视频最大 **500MB**
- 视频保留期限：**90天**（自动清理）
