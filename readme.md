# 基于音频信息的视觉显著物体检测设计与实现
## 毕业设计：Qingjiang Li（Hefei University of Technology）
前端：Vue

后端：SpringBoot

数据库：MySQL



## 前端运行：

```sh
cd pavsod_front
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
