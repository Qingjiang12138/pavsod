### 概述
本数据库设计用于支持视频显著性检测系统，包含用户管理、视频管理、检测任务管理等核心功能模块。

### 数据库设计
1. 用户表（user）

存储系统用户的基本信息和账户状态。

| 字段名      | 类型           | 约束                           | 说明             |
| -------- | ------------ | ---------------------------- | -------------- |
| id       | INT/BIGINT   | PRIMARY KEY, AUTO\_INCREMENT | 用户唯一标识         |
| 用户名      | VARCHAR(50)  | NOT NULL, UNIQUE             | 用户登录名          |
| 密码       | VARCHAR(255) | NOT NULL                     | 加密后的密码         |
| 头像       | VARCHAR(255) | NULL                         | 头像图片URL        |
| 邮箱       | VARCHAR(100) | NULL, UNIQUE                 | 用户邮箱地址         |
| 手机号码     | VARCHAR(20)  | NULL, UNIQUE                 | 用户手机号码         |
| 检测次数     | INT          | DEFAULT 0                    | 用户累计检测次数       |
| 累计处理视频数  | INT          | DEFAULT 0                    | 用户累计处理的原视频数量   |
| 累计检测2d视频 | INT          | DEFAULT 0                    | 累计检测的2D视频数量    |
| 累计检测3d视频 | INT          | DEFAULT 0                    | 累计检测的3D视频数量    |
| 目前存储空间   | BIGINT       | DEFAULT 0                    | 当前已使用的存储空间（字节） |
| 最大存储空间   | BIGINT       | DEFAULT 0                    | 用户存储空间上限（字节）   |
| 注册时间     | DATETIME     | DEFAULT CURRENT\_TIMESTAMP   | 账户注册时间         |

主键：id
唯一索引：用户名、邮箱、手机号

2. 原视频表（original_video）

存储用户上传的原始视频信息。

| 字段名   | 类型                  | 约束                           | 说明                       |
| ----- | ------------------- | ---------------------------- | ------------------------ |
| 视频id  | INT/BIGINT          | PRIMARY KEY, AUTO\_INCREMENT | 视频唯一标识                   |
| 视频名称  | VARCHAR(255)        | NOT NULL                     | 视频文件名                    |
| 视频类型  | TINYINT/VARCHAR(10) | NOT NULL                     | 视频格式类型（如：2d/3d/mp4/avi等） |
| 视频时长  | INT                 | NOT NULL                     | 视频时长（秒）                  |
| 视频大小  | BIGINT              | NOT NULL                     | 视频文件大小（字节）               |
| 视频url | VARCHAR(500)        | NOT NULL                     | 视频存储路径/访问URL             |
| 视频封面  | VARCHAR(500)        | NULL                         | 视频封面图片URL                |
| 上传时间  | DATETIME            | DEFAULT CURRENT\_TIMESTAMP   | 视频上传时间                   |

主键索引：视频id

3. 显著性检测视频表（salient_video）

存储经过显著性检测处理后生成的视频信息。

| 字段名     | 类型           | 约束                           | 说明           |
| ------- | ------------ | ---------------------------- | ------------ |
| 显著性视频id | INT/BIGINT   | PRIMARY KEY, AUTO\_INCREMENT | 显著性视频唯一标识    |
| 原视频id   | INT/BIGINT   | NOT NULL, FOREIGN KEY        | 关联的原视频ID     |
| 视频名称    | VARCHAR(255) | NOT NULL                     | 处理后视频文件名     |
| 视频时长    | INT          | NOT NULL                     | 视频时长（秒）      |
| 视频大小    | BIGINT       | NOT NULL                     | 视频文件大小（字节）   |
| 视频url   | VARCHAR(500) | NOT NULL                     | 视频存储路径/访问URL |
| 视频封面    | VARCHAR(500) | NULL                         | 视频封面图片URL    |


主键索引：显著性视频id

外键索引：原视频id → original_video(视频id)

关系说明： 与原视频表为一对一关系（1:1），一个原视频对应一个显著性检测结果视频。

4. 检测任务表（task）

存储视频显著性检测任务的执行信息和状态。

| 字段名     | 类型                  | 约束                           | 说明                    |
| ------- | ------------------- | ---------------------------- | --------------------- |
| 任务id    | INT/BIGINT          | PRIMARY KEY, AUTO\_INCREMENT | 任务唯一标识                |
| 用户id    | INT/BIGINT          | NOT NULL, FOREIGN KEY        | 创建任务的用户ID             |
| 原视频id   | INT/BIGINT          | NOT NULL, FOREIGN KEY        | 待检测的原视频ID             |
| 显著性视频id | INT/BIGINT          | NULL, FOREIGN KEY            | 生成的显著性视频ID（任务完成后填充）   |
| 视频类型    | TINYINT/VARCHAR(10) | NOT NULL                     | 检测视频类型（2d/3d）         |
| 目标帧率    | INT                 | NULL                         | 检测处理的目标帧率             |
| 进程数     | INT                 | DEFAULT 1                    | 任务使用的进程数量             |
| 任务状态    | TINYINT/VARCHAR(20) | NOT NULL                     | 任务状态（待处理/处理中/已完成/失败等） |
| 开始时间    | DATETIME            | NULL                         | 任务开始执行时间              |
| 结束时间    | DATETIME            | NULL                         | 任务完成或失败时间             |
| 错误信息    | TEXT                | NULL                         | 任务失败时的错误描述            |

主键索引：任务id

外键索引：

  用户id → user(id)\
  原视频id → original_video(视频id)\
  显著性视频id → salient_video(显著性视频id)

关系说明：

  与用户表：多对一关系（n:1），一个用户可创建多个任务\
  与原视频表：多对一关系（n:1），一个原视频可被多次检测（不同参数）\
  与显著性视频表：一对一关系（1:1），一个任务生成一个结果视频
