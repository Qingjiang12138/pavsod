### 概述
本数据库设计用于支持视频显著性检测系统，包含用户管理、视频管理、检测任务管理等核心功能模块。

# PavSod 数据库设计文档

**数据库名称**: `pavsod_db`  
**数据库版本**: MySQL 8.0.45  
**字符集**: utf8mb4  
**排序规则**: utf8mb4_0900_ai_ci

---

## 1. 用户表 (user)

存储用户基本信息及统计信息

| 字段名 | 字段类型 | 约束 | 说明 |
|--------|---------|------|------|
| `id` | bigint | **主键**、自增、降序索引 | 用户ID，系统自动生成 |
| `username` | varchar(50) | **非空**、唯一索引 | 用户名，用于登录和展示 |
| `password` | varchar(255) | **非空** | 登录密码（建议存储加密后的值） |
| `avatar` | varchar(255) | 可空，默认NULL | 用户头像URL地址 |
| `email` | varchar(100) | 可空，唯一索引 | 用户邮箱地址 |
| `phone` | varchar(20) | 可空，唯一索引 | 手机号码 |
| `detection_count` | int | 可空，默认0 | 累计检测次数统计 |
| `total_processed_videos` | int | 可空，默认0 | 累计处理视频总数 |
| `total_2d_videos` | int | 可空，默认0 | 累计检测2D视频数量 |
| `total_3d_videos` | int | 可空，默认0 | 累计检测3D视频数量 |
| `current_storage` | bigint | 可空，默认0 | 当前已用存储空间（字节） |
| `max_storage` | bigint | 可空，默认10 | 最大可用存储空间（字节），默认10字节 |
| `created_at` | datetime | 可空，默认CURRENT_TIMESTAMP | 用户注册时间 |

**索引说明**:
- 主键：`id` (降序)
- 唯一索引：`username`、`email`、`phone`

---

## 2. 原视频表 (original_video)

存储用户上传的原始视频信息

| 字段名 | 字段类型 | 约束 | 说明 |
|--------|---------|------|------|
| `video_id` | bigint | **主键**、自增 | 视频唯一标识ID |
| `video_name` | varchar(255) | **非空** | 原始视频文件名 |
| `video_type` | varchar(10) | **非空** | 视频类型，如：2d、3d等 |
| `duration` | int | **非空** | 视频时长（秒） |
| `video_size` | varchar(50) | **非空** | 视频文件大小（字节），字符串格式存储 |
| `video_url` | varchar(500) | **非空** | 视频文件存储URL地址 |
| `video_cover` | varchar(500) | 可空，默认NULL | 视频封面图片URL地址 |
| `upload_time` | datetime | 可空，默认CURRENT_TIMESTAMP | 视频上传时间 |

---

## 3. 显著性检测视频表 (salient_video)

存储经过显著性检测处理后生成的视频

| 字段名 | 字段类型 | 约束 | 说明 |
|--------|---------|------|------|
| `salient_video_id` | bigint | **主键**、自增 | 显著性视频唯一标识 |
| `original_video_id` | bigint | **非空**、唯一索引、**外键** | 关联的原视频ID，一对一关系 |
| `video_name` | varchar(255) | **非空** | 处理后视频文件名 |
| `duration` | int | **非空** | 视频时长（秒） |
| `video_size` | bigint | **非空** | 视频文件大小（字节），数值格式存储 |
| `video_url` | varchar(500) | **非空** | 处理后视频存储URL |
| `video_cover` | varchar(500) | 可空，默认NULL | 视频封面URL |

**外键约束**:
- `original_video_id` → `original_video(video_id)`，级联删除

---

## 4. 检测任务表 (task)

记录视频显著性检测任务的状态和进度

| 字段名 | 字段类型 | 约束 | 说明 |
|--------|---------|------|------|
| `task_id` | bigint | **非空**、自增、唯一索引 | 任务唯一标识ID |
| `user_id` | bigint | **主键**（复合）、**非空**、**外键** | 创建任务的用户ID |
| `original_video_id` | bigint | **主键**（复合）、**非空**、**外键** | 待处理的原视频ID |
| `salient_video_id` | bigint | 可空，默认NULL、**外键** | 处理完成后的显著性视频ID |
| `video_type` | varchar(10) | **非空** | 视频类型（2d/3d） |
| `target_fps` | int | 可空，默认NULL | 目标处理帧率 |
| `process_count` | int | 可空，默认1 | 处理使用的进程数 |
| `task_status` | int | **非空**，默认0 | 任务状态：0-待处理，1-处理中，2-已完成，3-失败 |
| `start_time` | datetime | 可空，默认NULL | 任务开始处理时间 |
| `end_time` | datetime | 可空，默认NULL | 任务完成时间 |
| `error_msg` | text | 可空 | 任务失败时的错误信息 |
| `create_at` | datetime | 可空，默认NULL | 任务创建时间（视频上传时间） |

**约束说明**:
- **复合主键**：`(user_id, original_video_id)` - 确保一个用户同一视频只能有一个任务
- **唯一索引**：`uk_task_id` (task_id)

**外键约束**:
- `user_id` → `user(id)`，级联删除
- `original_video_id` → `original_video(video_id)`，级联删除
- `salient_video_id` → `salient_video(salient_video_id)`，删除时设为NULL

**索引**:
- `idx_task_id`: task_id索引
- `idx_salient_video_id`: 显著性视频ID索引
- `idx_task_status`: 任务状态索引（便于按状态查询）
- `idx_start_time`: 开始时间索引
